# Challenge 1: Amigo Secreto

Este proyecto es el primer desafío del programa de Formación para Principiantes en Programación de Oracle Next Education - Alura Latam. Consiste en desarrollar una aplicación que permite a los usuarios ingresar nombres y realizar un sorteo aleatorio de uno de los nombres ingresados.

## Descripción

La aplicación "Amigo Secreto" facilita la organización de sorteos de regalos o intercambios de forma sencilla y divertida. Los usuarios ingresan los nombres de los participantes y la aplicación se encarga de seleccionar un ganador al azar.

## Funcionalidades

*   **Validación de campos vacíos:** Se muestra una alerta en pantalla si el usuario intenta realizar el sorteo sin haber ingresado ningún nombre.
*   **Control de nombres duplicados:** Si un usuario ya se encuentra en la lista, la aplicación indica que el usuario ya existe y no lo inserta nuevamente.
*   **Sorteo aleatorio:** La aplicación selecciona un nombre al azar de la lista de participantes para determinar el ganador.

## Tecnologías utilizadas

*   HTML
*   CSS
*   JavaScript

## Instrucciones de uso

1.  Ingresa los nombres de los participantes en el campo correspondiente.
2.  Haz clic en el botón "Sortear" para realizar el sorteo.
3.  El nombre del ganador se mostrará en pantalla.

## Próximas mejoras

*   Interfaz de usuario más atractiva: Mejorar el diseño y la experiencia de usuario.
*   Persistencia de datos: Implementar una forma de guardar la lista de participantes para sorteos futuros.
*   Funcionalidad de "Amigo Secreto" tradicional: Permitir asignar un amigo secreto a cada participante, en lugar de solo un ganador.
